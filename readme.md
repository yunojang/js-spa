# vanilaJs - spa

practice vanilaJs(actually ts), SPA(single page application)

## product

- server: express
- client: ts
  - build: webpack, babel

## run

- server dev (express)

```
  yarn dev
```

- server run

```
yarn start
```

- client dev

```
cd frontend
yarn start
```

- client build

```
cd frontend
yarn build
```
